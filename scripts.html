Scripts Ricardo

1. Baixar o Google Chrome e modificar o atalho

wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
sudo gdebi google-chrome*

Modificar o atalho e adicionar aos favoritos na barra.

/usr/bin/google-chrome-stable %U --password-store=basic --incognito https://educacao.pr.gov.br/iniciar

2. Autorizar convidado e reiniciar o computador

Baixe o script modificado.

wget https://ricardobranco.vercel.app/scriptRemoverProgramas.sh
chmod +x scriptRemoverProgramas.sh
sudo ./scriptRemoverProgramas.sh

3. Testando - Ativar a conta de convidado (se necessário)

Se o script não ativar a conta de convidado, use:

sudo xed /etc/lightdm/lightdm.conf

Adicionar as seguintes linhas no arquivo:

autologin-guest=true
autologin-user-timeout=0
allow-guest=true

4. Copiar a pasta para os outros perfis

sudo cp -a ~/.bashrc ~/.profile ~/.config /etc/skel/
Comando antigo: 
sudo cp -a ~/.* /etc/skel/
